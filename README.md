# SBI_MITRA_Demo
SBI MITRA is an Agentic AI system that acts as a proactive banking companion — intelligently acquiring new customers, driving adoption of digital services, and engaging existing customers through life-event based personalization.
Unlike traditional chatbots that only respond, MITRA acts autonomously. It detects behavioral and financial signals, decides the right moment to reach out, selects the most relevant product, and initiates a natural conversational interaction — all without waiting for the customer to ask.
Three core agent loops:

Acquire Agent — Identifies high-intent prospects via behavioral signals and guides them through conversational, vernacular-friendly KYC onboarding
Adopt Agent — Post-onboarding, nudges customers toward UPI, SIPs, insurance, FDs based on their financial profile and usage gaps
Engage Agent — Monitors life events (first salary, EMI completion, travel spend, large purchases) and sends hyper-personalized, contextually timed outreach
